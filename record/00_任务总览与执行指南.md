## 一、项目简介

本项目旨在将三份独立调研草稿合并为一份完整的中文可行性研究报告（docx格式），主题为"中国饮品品牌进入加拿大安大略省市场的可行性"。第一版合并版报告已由 Claude 自动生成，但经评审发现两个显著不足：（1）对近年新进入加拿大市场的中国茶饮/咖啡品牌分析不够深入；（2）定量研究的数据链条不完整，财务论证说服力不足。因此新增 Phase-0 深化研究阶段，在审校（Phase-1）之前先补强内容。

## 五、质量检查点（Gate）

### Gate 0：Phase-0 完成后

在启动 Phase-1 之前，项目负责人需确认：
- [ ] Phase-0a 品牌情报覆盖了至少 HEYTEA、茶百道、库迪、瑞幸、霸王茶姬、蜜雪冰城 6个品牌
- [ ] Phase-0b 完成了 TAM→SAM→SOM 推导链，且日均订单量经过至少两种方法交叉验证
- [ ] Phase-0b 包含完整的单店P&L损益表（三种情景）
- [ ] Phase-0c 已将新内容整合进报告docx，新增章节（2.5、4.1.1、8.0、8.4等）已写入

### Gate 1：Phase-1 全部完成后

在启动 Phase-2 之前，项目负责人需确认：
- [ ] Phase-1a 审校意见已输出，无"高严重"遗漏项未被标记
- [ ] Phase-1d 冲突决策已有推荐方案
- [ ] Phase-1b 排版问题已记录
- [ ] Phase-1c 参考文献核实已完成，失效链接已标记

### Gate 2：Phase-2 完成后

在启动 Phase-3 之前，项目负责人需确认：
- [ ] 6–8张图表已生成，数据与报告正文一致
- [ ] 图表使用的数据来源已在清单中注明

### Gate 3：Phase-3 完成后（最终交付）

- [ ] 终版docx在Word中打开正常，目录可更新
- [ ] 所有图表在文档中正常显示
- [ ] 参考文献中无失效链接（或已标注）
- [ ] 精简版执行摘要与终版数据一致
- [ ] record文件夹下所有过程文档齐全
