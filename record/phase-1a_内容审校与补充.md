我们正在为一家中国饮品企业撰写"进入加拿大安大略省饮品市场的可行性研究报告"。目前已有三份独立草稿，并已用 Claude 将三份草稿合并为一份综合版报告（docx格式）。

你的任务是：**逐章比对合并版报告与三份原始草稿，找出遗漏、错误和不一致之处，输出审校意见清单。**

## 三、具体执行步骤

### 步骤 1：提取合并版报告全文

用 pandoc 将合并版 docx 转为 markdown，方便逐章阅读：
```bash
pandoc "安大略省饮品市场进入可行性研究报告（合并版）.docx" -o 合并版_全文.md
```

### 步骤 2：逐章审校

按以下章节逐一比对合并版与三份原始草稿，检查以下要点：

**（一）数据完整性检查**
- 三份草稿中出现的关键数据点（市场规模、增长率、人口数据、投资金额、成本数据等）是否都在合并版中体现
- 特别注意以下容易遗漏的内容：
  - 草稿一中的：五年现金流/IRR模型、敏感性分析热力图描述、竞争者对比表（含Tim Hortons / Starbucks / Second Cup / Booster Juice / Jugo Juice等10+品牌）
  - 草稿二中的：万锦市Zoning停车位配比的具体数字（每100平方米6.67个）、WSIB保费降至$1.38/100加元工资、饭团外卖（Fantuan）策略、小红书营销策略、中央厨房+卫星店模式
  - 草稿三中的：HEYTEA 2026年2月Eaton Centre开店细节、Chagee（霸王茶姬）九段线争议、纽芬兰省糖税废除（2025年7月）、加拿大-中国税收协定（1986年）、AODA无障碍法案

**（二）数字准确性检查**
- 所有金额（CAD）、百分比、人口数、门店数是否与原始草稿一致
- 特别核对以下高敏感数据：
  - 珍珠奶茶市场规模：2025年约$1亿–$1.4亿CAD
  - 盈亏平衡点：标准店~107杯/日
  - 安省最低工资：$17.60/小时（2025年10月起）
  - 多伦多餐饮执照：$516 CAD
  - SFC进口许可：$299.86 CAD
  - HEYTEA定价上限：$12.90 CAD
  - 加拿大餐厅亏损比例：62%

**（三）引用标注完整性**
- 每个关键数据点后面是否有引用来源标注（如 "Statistics Canada, 2021" 或 "CBRE, 2025"）
- 标注是否与参考文献列表对应

**（四）逻辑一致性**
- 执行摘要中的结论是否与正文分析一致
- 冲突对照表（附录A）中列出的8项冲突是否在正文中都有标注提醒

### 步骤 3：检查图片/图表引用

草稿二原文中包含两处图片引用（`media/image1.png` 和 `media/image2.png`），分别是：
- image1: 盈亏平衡点相关的图表
- image2: 品牌策略相关的图表

检查合并版中是否对这两处做了处理（应改为文字描述，注明"原草稿二含图表，此处以文字说明替代"）。

## 四、注意事项

1. **只审校，不修改**——你的任务是发现问题并记录，不需要修改合并版报告本身
2. **以中文输出**所有审校意见
3. **标注严重程度**：高（影响结论或决策）、中（数据不精确但不影响方向）、低（措辞/格式问题）
4. 如果发现合并版中存在三份草稿都没有的"新增内容"，也请标注出来（可能是合并过程中产生的推断）
