### 图表6：盈亏平衡敏感性热力图

**数据来源**：报告第八章 8.3节

构建一个5×5的矩阵：
- X轴：日均订单量（100, 150, 200, 250, 300）
- Y轴：净租金（$/sqft/年）（30, 40, 50, 60, 70）
- 数值：回本月数（根据固定成本和毛利公式推算）

**简化计算公式**：
- 月营收 = 日均杯量 × 30 × $7.50
- 月毛利 = 月营收 × 75%
- 月固定成本 = 租金 + $9,000(人工基准) + $2,000(其他)
- 月租金 = 净租金($/sqft/年) × 900(sqft) / 12
- 月净利 = 月毛利 - 月固定成本
- 回本月数 = $350,000(标准店投资) / 月净利（如月净利≤0则标记为"N/A"）

**样式要求**：
- 热力图（红=回本慢/不可行，绿=回本快）
- 每个格子内标注回本月数
- 标题：标准店投资回本期敏感性分析（月）
- 中文标签

## 四、技术要求

### 推荐工具
- **Python + matplotlib**（推荐，中文支持好）
- 备选：Python + plotly（交互性更强，但需导出为静态图）

### 中文字体设置（matplotlib）
```python
import matplotlib.pyplot as plt
import matplotlib
# 尝试使用系统中文字体
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'WenQuanYi Micro Hei', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False
```

### 输出格式
- 格式：PNG，300 DPI
- 文件命名：`chart_01_市场规模对比.png`、`chart_02_市场增长预测.png`……
- 保存位置：项目根目录下新建 `charts/` 文件夹

### 配色方案（建议）
- 主色调：深蓝(#2E75B6)、浅蓝(#9DC3E6)
- 辅助色：深绿(#548235)、橙色(#ED7D31)、灰色(#A5A5A5)
- 警示色：红色(#FF0000)用于风险/亏损区域
- 背景色：白色(#FFFFFF)
