## 三、任务A：生成终版完整报告

### 步骤1：整理修改清单

汇总所有Phase的发现，按以下分类编制修改清单：

**（一）内容修改（来自Phase-1a）**
- 需要补充的遗漏内容
- 需要修正的错误数据
- 需要补充的引用标注

**（二）冲突项处理（来自Phase-1d）**
- 对报告正文中的"[冲突提示]"标注进行更新
- 在每个冲突点旁注明"决策备忘录推荐方案"
- 更新附录A冲突对照表中的"建议处理方式"列

**（三）参考文献修改（来自Phase-1c）**
- 移除或替换失效URL
- 修正标题/年份不匹配的条目
- 统一APA格式

**（四）排版修改（来自Phase-1b）**
- 修复表格显示问题
- 修正分页位置
- 调整标题/页眉/页脚

**（五）图表嵌入（来自Phase-2）**
- 在对应章节插入图表

### 步骤2：修改docx生成脚本

报告的生成脚本位于临时目录下（`generate_report.js`，使用Node.js + docx-js库）。根据修改清单，更新脚本中的内容。

主要修改点包括：
1. **补充遗漏内容**：在对应章节的Paragraph数组中添加新段落
2. **修正错误数据**：搜索并替换脚本中的错误数值
3. **嵌入图表**：使用ImageRun添加PNG图片（参考以下代码模板）

```javascript
// 图表嵌入模板
const fs = require("fs");
const { ImageRun, Paragraph } = require("docx");

// 在对应章节的children数组中添加：
new Paragraph({
  children: [
    new ImageRun({
      type: "png",
      data: fs.readFileSync("charts/chart_01_市场规模对比.png"),
      transformation: { width: 550, height: 350 }, // 根据实际图表比例调整
      altText: {
        title: "图1：加拿大核心饮品品类市场规模",
        description: "加拿大核心饮品品类市场规模对比柱状图",
        name: "chart_01"
      }
    })
  ],
  alignment: "center"
}),
new Paragraph({
  children: [new TextRun({ text: "图1：加拿大核心饮品品类市场规模（2024/2025）", size: 20, italics: true })],
  alignment: "center",
  spacing: { after: 240 }
}),
```

4. **更新参考文献**：替换或移除失效条目
5. **修复排版问题**：调整表格列宽、分页符位置等

## 四、任务B：生成精简版执行摘要

### 目标

制作一份2–3页的独立精简版报告，供高管快速阅读决策。不需要详细数据论证，只需核心结论和关键数字。

### 内容结构

```
标题：加拿大安大略省饮品市场进入可行性研究——执行摘要

一、核心结论（1段，约150字）
- 是否进入：是
- 推荐模式：先直营后加盟
- 投资规模：标准店$24.5万–$43.5万CAD
- 回本预期：17–31个月

二、市场机遇（1段+1个迷你表格）
- 市场规模与增长率
- 人口基础
- 差异化空白

三、关键财务指标（1个精简表格）
- 盈亏平衡：107杯/日
- 目标净利率：10–20%
- 毛利率：75–88%

四、推荐行动计划（3个阶段，各1–2句话）
- Phase 1：GTA 2–3家直营
- Phase 2：扩至5–8家
- Phase 3：直营+加盟

五、TOP 3 风险（各1句话）
1. 核心区域市场饱和
2. 劳动力短缺与成本上涨
3. 供应链对亚洲进口依赖

六、下一步行动建议（3条具体行动项）
1. 确定冲突项的最终决策（见冲突决策备忘录）
2. 启动GTA选址考察
3. 开始企业注册流程
```

### 格式要求

- 同样使用 docx-js 生成或手动在 Word 中创建
- A4纸，Arial字体
- 标题16号加粗，正文12号
- 可使用简洁的表格和加粗来突出关键数字
- 不需要参考文献（正文版已有完整引用）
- 文末注明："详细数据论证及完整参考文献请见《安大略省饮品市场进入可行性研究报告（终版）》"

### 输出文件

保存为：`安大略省饮品市场研究_执行摘要.docx`
